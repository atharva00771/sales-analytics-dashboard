# 📊 Sales Analytics Dashboard | Power BI Project

## 📌 Project Overview
The Sales Analytics Dashboard is an interactive business intelligence project developed using Microsoft Power BI. The dashboard provides comprehensive insights into sales performance, profitability, product performance, and customer trends through interactive visualizations and key performance indicators (KPIs).

This project helps organizations monitor business performance, identify top-performing products, analyze sales trends, and make data-driven decisions.

---

## 🎯 Project Objectives
- Analyze overall sales performance and profitability.
- Track key business metrics using KPI cards.
- Identify top-performing products and categories.
- Monitor monthly sales trends and growth.
- Generate business insights through interactive dashboards.
- Support decision-making with data visualization.

---

## 🛠 Tools & Technologies Used
- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Data Modeling
- Data Visualization
- Microsoft Excel

---

## 📂 Dataset Information
The dataset contains information related to:
- Product Name
- Product Category
- Sales
- Profit
- Cost
- Segment
- Country
- Units Sold
- Order Date
